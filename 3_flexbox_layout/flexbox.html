The flex-container properties and values

flex-direction

The flex-direction property has the following values: column; column-reverse; row; row-reverse;

flex-wrap

The flex-wrap property has the following values: wrap; nowrap; wrap-reverse;

flex-flow

This is the short-hand property for flex-direction and flex-wrap;

justify-content

The justify-content property has the following values: center; flex-start; flex-end; space-between; space-around;

align-items

The align-items property has the following values: center; flex-start; flex-end; stretch; baseline;

align-content

The align-content property has the following values: space-between; space-around; stretch; center; flex-start; flex-end; . For perfect centering set both the justify-content and align-items properties to center, then the flex item will be perfectly centered.

CSS flex item properties.

order

The order property specifies the order of the flex items. It can change the order of the flex items.

flex-grow

The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items. The value must be a number, default value is 0

flex-shrink

The flex-shrink property specifies how much a flex item will shrink relative to the rest of the flex items. The value must be a number, default value is 1.

flex-basis

The flex-basis property specifies the initial length of a flex item.

flex

The flex property is a shorthand property for the flex-grow, flex-shrink, and flex-basis properties.
